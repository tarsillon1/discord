# AWS Lambda Discord

An an example of a Discord webhook integration using AWS lambda.

## Getting Started

1. Install project dependencies:

```bash
$ npm i
```

2. Build the lambda:

```bash
$ npm run build
```

3. Package the lambda:

```bash
$ npm run package
```

4. Deploy the lambda to AWS:

```bash
$ PUBLIC_KEY=$YOUR_DISCORD_APP_PUBLIC_KEY npm run deploy
```
